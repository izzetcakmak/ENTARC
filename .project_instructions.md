# ENTARC — Autonomous Venture & Market Intelligence Agent on Arc Network

## Project Overview
ENTARC is an autonomous VC AI agent for pre-TGE project discovery, trust scoring, and investment management on Arc Network. Built for the **Programmable Money Hackathon · Build on Arc** (Agentic Track · Final Submission: August 9, 2026 · Demo Day: August 20, 2026).

## Hackathon Strategy
- **Track:** Agentic Track — autonomous agents transacting with programmable money (USDC on Arc)
- **Final requirements:** Working MVP on Arc + public repo + 3-minute video + presentation deck

## Technical Architecture
- Dark cyberpunk/glassmorphism design with slate-900 backgrounds, cyan/violet accents
- GlassCard component for all card-based UI
- `(main)` route group contains authenticated pages with sidebar layout
- Pages: Dashboard, Discovery, Portfolio, Agent Hub, Autonomous Agent, AI Insights, Settings, Submit Project

## Key Integrations
- **Circle Agent Stack:** Agent wallets (`@circle-fin/developer-controlled-wallets`), milestone escrow
- **Agent Wallet:** `0x72f5adfabc8670bc873963018133d0b1729a92fd` (ARC-TESTNET, LIVE)
- **Circle API:** TEST keys configured in `.env` — **Mainnet Wallets access approved** (can deploy to mainnet)
- **Arc Network:** Custom chain config (chainId: 1868), L2 on Ethereum

## API Routes (Hackathon)
- `/api/agent/portfolio-manager` — Adaptive Portfolio Manager: risk regime detection, rebalancing
- `/api/agent/signal-aggregator` — Signal Aggregator: multi-source signal analysis
- `/api/agent/escrow` — Circle escrow: milestone-based USDC management
- `/api/circle/wallets` — Agent wallet creation/listing
- `/api/circle/status` — Circle integration health check
